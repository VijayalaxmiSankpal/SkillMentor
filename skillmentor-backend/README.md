# SkillMentor

Full Stack Career & Interview Preparation App

---

## 🚀 Project Structure

### Backend
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- Gemini AI integration
- Nodemailer (email service)

### Frontend
- React + Vite
- Tailwind CSS
- Context API / Hooks

---

## ⚙️ Setup (Backend)

```bash
npm install
cp .env.example .env
npm run dev
